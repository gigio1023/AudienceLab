# Dashboard

React/Vite frontend for monitoring the simulation in real time and reviewing evaluation snapshots. The UI assumes a pre-selected influencer shortlist (search is out of scope).

## What It Shows

- Live simulation status from `shared/simulation/{simulationId}.json`
- Agent activity streams from `public/simulation/*.jsonl`
- Evaluation snapshot from `public/evaluation/latest.json`

## Quick Start

```bash
npm install
npm run dev
```

Open the Vite URL (typically `http://localhost:5173`).

## Data Inputs

### Simulation status
- `shared/simulation/{simulationId}.json`
- Polled by the dashboard hook

### Agent activity feed
- `dashboard/public/simulation/*.jsonl`
- Optional index file: `dashboard/public/simulation/__files.json`

### Evaluation snapshot
- `dashboard/public/evaluation/latest.json`
- Generated by `eval-agent/evaluate.py`

## Environment Variables

- `VITE_SIMULATION_ID` (default: `latest`)
- `VITE_AGENT_FEEDS` (comma-separated list of JSONL files)
- `VITE_EVALUATION_ID` (default: `latest`)
